# StudentHub

StudentHub is a lightweight, cloud-native Student Management REST API built with FastAPI, MySQL, Docker, and Kubernetes. It provides CRUD operations for managing student records, health monitoring endpoints, containerized deployment, and scalable orchestration support, making it an ideal reference project for learning modern backend development and DevOps practices.

# Key Features
RESTful CRUD APIs for student management
Health check endpoint
MySQL database integration
Environment-based configuration
Docker containerization
Kubernetes deployment support
Simple and extensible project structure

# Tech Stack: 
Python, FastAPI, MySQL, SQLAlchemy, Docker, Kubernetes.
